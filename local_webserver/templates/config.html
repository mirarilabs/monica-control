How to Find Pico IP Address

On your Pico, when you run the command server, it will display:

Command server started at 192.168.1.XXX:8080

Use that IP address in the field above.

Setup Instructions

  1. On the Pico: Run python main.py and select mode with command server
  2. On this computer: Run python local_web_server.py
  3. Update IP: Enter the Pico's IP address above and click "Update"
  4. Test: Click "Test Connection" to verify communication
  5. Control: Go to the main interface to control Monica
